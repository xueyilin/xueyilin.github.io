OS Cloud App
Dealing with Complex System Design

Role: Product Designer & Manager | Information Architect

Team Size: 8      Date: 2015/10-2015/1

Category: Interaction DesignMobile App Information Architecture

Intro

Thunder OS is a ROM based on android that is free to download and install through 360 PC Security software. It provides solution for low price smart phones to maintain fast speed and long battery life.

It is made available for around 800 different android devices and can be easily installed just through one click.

Goal

Design a cloud service app that synchronizes the contact books around all devices and build an account system that will later be applied to all the product line of this ROM.

Challenge

The main challenge for this project is defining the inner logic under the synchronized contacts and automatic backup features such as how and when to backup data. This also includes considering solution for edge cases such as when device is under low battery, bad internet connection, cellular network, or when the server is broken.

Another component that I have to tackle is a simple user registration and login process to have more people onboard and be reusable for all account related features in this ROM.

My Role

As the product designer, I define features of the product, design user experience and information architecture. I also work closely with 1 UI designer to ensure the interface delivers the brand and stays coherent with the whole ecosystem.

As the only product manager in charge of this project, I coordinate developers, designers, and QAs to ensure agile development process. I work with marketing team on how to use this product and the feature it provides to promote the ROM.

The product was projected to launch in a month and half, so the main pressure is to make sure it passes testing on over 100 different android devices on market before launching.

Key Features

Synchronized Contact Book: Allow users to have the same version of contacts through all devices using the same Thunder OS account

Contacts Time Machine: Keep record of all of user's contacts version, so that they can retrieve deleted or lost contacts.

Automatic Data Back-up: Automatically back-up user's messages, photos, phone logs and app data.

Device Change Helper: Users can manually back-up and retrieve data such as app list, launcher arrangement, wifi settings, just by one click.

Fast App Backup: Allow users to back-up all their selected apps in less than one minute.
